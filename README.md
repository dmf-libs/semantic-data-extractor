## Goals

The initial main recognition goals are:

- Boolean [Done]
- URL [Done]
- DateTime [Done]
- Integer number [Done]
- Float number [Done]
- Image URL
- Geo Location
- File
- Document
- Presentation
- Spreadsheet
- City
- Country
- Person Name
- Person Title
- Currency
- BG phone number
- BG Address
